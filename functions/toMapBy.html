  • Creates a map from the own entries of an object.

    Since

    13.0.0

    See

    toMap

    Returns

    Map created from the object.

    Example

    toMapBy({a: 1, b: 4, c: 5}, key => { return { key }; }, value => value * 2)
    // => Map{{key: "a"}: 2, {key: "b"}: 8, {key: "a"}: 10}

    toMapBy([1, 2, 3], (_key, value) => value, (_key, value) => value * 2)
    // => Map{1: 2, 2: 4, 3: 6}
