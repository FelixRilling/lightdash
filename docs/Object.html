(static) objDecycle(obj, fnopt, referencesopt) → {Object}

Source:
  • lightdash.esm.js, line 1039
Since:
  • 6.0.0
Replaces every cyclic reference of an object. Can take a custom replacer function and a pre-filled WeakSet of references.
Example
const a = {b: 1, c: 2};

a.a = a;

objDecycle(a)
// => {a: null, b: 1, c: 2}

objDecycle(a,key=>`_${key}`)
// => {a: "_a", b: 1, c: 2}
