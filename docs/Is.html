Checks if the string is blank (no non-space content).

Example
isBlank("")
// => true

isBlank("  ")
// => true

isBlank(" foo ")
// => false

Source:

Checks if a value is a promise.

Example
isPromise(new Promise((resolve, reject) => resolve("foo")));
// => true

isPromise(() => "foo");
// => false
