(static) isEmpty(val) → {boolean}

Source:
  • lightdash.esm.js, line 376
Since:
  • 1.0.0

Checks if a value is empty

A value is consider empty if it is either a primitive or an object-like without content

Array-likes are considered empty if they have a length of zero, set/maps if they have a size of zero, and objects if their keys have a length of zero

Examples
//returns true
isEmpty([])
isEmpty({})
isEmpty("")
isEmpty(true)
isEmpty(1)
isEmpty(null)
//returns false
isEmpty("foo")
isEmpty([1, 2])
isEmpty({a: 1})
