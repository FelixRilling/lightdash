(static) hasPath(target, path) → {boolean}

Source:
  • lightdash.esm.js, line 716
Since:
  • 2.0.0
Checks if a target has a path-array of key-strings.
Examples
hasPath({a: 1}, ["a"]);
// => true

hasPath([4, 6, 8], ["1"]);
// => true

hasPath({a: {b: 2, c: [10, 20]}}, ["a", "c", "0"]);
// => true
hasPath({a: 1}, ["c"]);
// => false

hasPath([4, 6, 8], ["8"]);
// => false

hasPath({a: {b: 2, c: [10, 20]}}, ["f", "x", "231", "21"]);
// => false

hasPath(1, ["foo"]);
// => false
