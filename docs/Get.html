(static) getPath(target, path) → {boolean}

Source:
  • lightdash.esm.js, line 612
Since:
  • 2.0.0
Accesses a target by a path-array of key-strings. If the path doesn't exist, null is returned.
Example
getPath({a: 1}, ["a"]);                              //=> 1
getPath([4, 6, 8], ["1"]);                           //=> 6
getPath({a: {b: 2, c: [10, 20]}}, ["a", "c", "0"]);  //=> 10
