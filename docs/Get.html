(static) getPath(target, path) → {null|any}

Source:
  • lightdash.esm.js, line 738
Since:
  • 2.0.0
Returns a targets value in a given path. If the path doesn't exist in the target, null is returned.
Example
getPath({a: 1}, ["a"]);
// => 1

getPath([4, 6, 8], ["1"]);
// => 6

getPath({a: {b: 2, c: [10, 20]}}, ["a", "c", "0"]);
// => 10

(static) getSize(val) → {number}

Source:
  • lightdash.esm.js, line 360
Since:
  • 6.0.0
Gets size of a value. If the value has a length or size property, the value of that is returned. If the value is object-like, the length of its entries is returned. Otherwise -1 is returned.
Example
getSize([1,2])
// => 2

getSize({a:1})
// => 1

getSize(new Set())
// => 0

isEmpty("foo")
// => 3

isEmpty(1)
// => -1

isEmpty(null)
// => -1
