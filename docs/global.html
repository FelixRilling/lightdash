decycle(collection, replacer, references)

Source:
  • lightdash.esm.js, line 214
Since:
  • 6.0.0

Replaces every circular reference in an object with a value, defaulting to null.

Can take a custom replacer function and a pre-filled weak set of references.

Example
const a = {a: {}, b: 1, c: 2};

a.a = a;

decycle(a)
// => {a: null, b: 1, c: 2}

decycle(a, key => `_${key}`)
// => {a: "_a", b: 1, c: 2}
Parameters:
Name Type Description
collection

Object to decycle.
replacer

Value replacer function
references

WeakSet prefilled with encountered references, not recommended to provide this manually.
Returns:

Decycled object.

name(value)

Source:
  • lightdash.esm.js, line 257
Since:
  • 10.2.0

Gets name of a value.

If the value has a name or description property, the value of that is returned. If the value is a string, it is returned as is. Otherwise null is returned.

Example
name(class Foo{})
// => "Foo"

name(function bar(){})
// => "bar"

name(Symbol("abc"))
// => "abc"

name("foo")
// => "foo"

name(1)
// => null

Name Type Description
value

Returns:

The name of the value.

similar(str, list, returnFull)

Source:
  • lightdash.esm.js, line 139
Since:
  • 6.3.0

Returns strings similar to the input based its distance to the values in the list given.

Example
similar("Fob", ["Foo", "Bar"])
// => ["Foo"]

similar("cmmit", ["init", "commit", "push"])
// => ["commit"]

similar("Kitten", ["Sitten", "Sitting", "Bitten"])
// => ["Sitten", "Bitten"]

similar("cmmit", ["init", "commit", "push"], true)
// => Map<number, string[]>{1: ["commit"], 3: ["init"], 5: ["push"]}
Parameters:

Returns:
